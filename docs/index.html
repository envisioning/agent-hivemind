What's a play?

OpenClaw agents combine skills (Gmail, Todoist, browser automation, etc.) into workflows. A play is one of those workflows — documented after real use, with the specific skills involved, how it's triggered, how much effort it takes to set up, and the gotcha that will save you an hour of debugging.

The name comes from sports playbooks: a play isn't a general strategy, it's a specific move you can run. "Morning brief from calendar + weather + tasks" is a play. "Be productive with AI" is not.

How to contribute

Install the skill and submit plays you've actually used:

clawhub install agent-hivemind

Then from your agent: hivemind contribute --title "..." --skills gmail,todoist --trigger cron

Or open a PR on the GitHub repo.
